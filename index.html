Download and try the game on Windows

ALL ABOUT THE MAZE PROJECT

We had to create our own portfolio project at the end of ALX/Holberton School foundational trimesters, Learning about game development sparked my interest in working on Maze-Game. I was interested in game development and wanted to learn about new processes and techniques that could be used in software development. ... To build this game, i decided to learn about the Raycasting tool and implement it using the SDL2 library. It's intriguing to learn how early games, such as Wolfenstein 3D, were able to adopt a pseudo-3D visual that transformed the industry and paved the way for other gaming engines and technologies. .

Keyboard Keys for Players

The keys below will help you to interact with the game

  • Arrow Keys -- Player Directions
  • M -- Map
  • R -- Refresh
  • C -- Zoom Player view
  • [ -- Zoom Out in Game
  • ] -- Zoom In in Game
  • F -- Change Views (Normal view to Panoramic view and vice versa)
  • T -- Change Texture of walls and square columns
